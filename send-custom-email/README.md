# send-custom-email
Send custom Email using HTML and CSS

## Getting Started
<pre>
1. git clone <paste the address here>
2. cd send-custom-email
3. npm install
4. Create a .env file and add the following:

    MAIL_ID=your-gmail-account@gmail.com
    MAIL_PASS=your-password
    
5. npm start 
</pre>
